Indicator OWL prototype aligned with AIAO, ClaimOnt, ImpactOnt, and InfoComm

IRI: http://independentimpact.org/indicator-owl/

Ontology for expressing indicator semantics, providing OWL descriptions aligned with aiao, claimont, impactont, and infocomm. While originally developed for SDG indicators, this ontology has general applicability to any indicator framework.

Ontology Dependencies

This ontology builds upon and conforms to the following foundational ontologies:

  • Impact Ontology (ImpactOnt) - Provides foundational semantic framework for impact modeling
  • Anthropogenic Impact Accounting Ontology (AIAO) - Provides semantic framework for accounting for human impact

These dependencies ensure semantic interoperability and consistency with established impact assessment frameworks.
